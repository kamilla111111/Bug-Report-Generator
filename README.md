# Bug Report Generator

Десктопное приложение для автоматического формирования сводки по багам из Google-таблицы.

---

## Что делает

- Читает данные из Google Sheets по ссылке
- Считает количество багов по критичности (Critical, Major, Minor)
- Показывает, сколько багов ещё открыто
- Сохраняет отчёт в текстовый файл

---

## Как работает

1. Вставьте ссылку на Google-таблицу
2. Нажмите «Сформировать отчёт»
3. Получите сводку по багам
4. Сохраните отчёт на компьютер
